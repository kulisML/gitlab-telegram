# GitLab Notification Bot

Бот для Telegram, который получает уведомления о событиях в GitLab через вебхуки и пересылает их в указанный чат.

## Основные возможности

- **Уведомления о пушах:** автор, ветка и список последних 5 коммитов.
- **Merge Requests:** открытие, слияние и закрытие MR с именами участников.
- **CI/CD Пайплайны:** оповещения об успешном завершении или ошибках в пайплайнах.
- **Комментарии:** мгновенные уведомления о новых обсуждениях в коде или MR.
- **Issues:** создание и закрытие задач.
- **Безопасность:** поддержка `Secret Token` для верификации запросов от GitLab.

### Быстрый старт

1. Настройте файл `.env` (см. `docs/setup.md`).
2. Запустите бота через Docker:
   ```bash
   docker-compose up -d
   ```
3. Добавьте Webhook в настройках проекта GitLab: `http://your-server-ip:8000/gitlab`.

Подробную инструкцию см. в [инструкции по настройке](./setup.md).

# Telegram-команды бота

Бот поддерживает интерактивное управление прямо из Telegram.
Напишите `/tgbot` в любом чате, где присутствует бот.

---

## Команда `/tgbot`

Главная команда. Открывает панель управления с инлайн-кнопками.

При вызове бот **автоматически**:
- Сохраняет ваш Telegram ID и username в базу данных
- Запоминает вас для будущих @упоминаний

```
🤖 GitLab Bot — Панель управления

👤 Вы: @your_username (ID: 123456789)
⚠️ GitLab username не привязан — нажмите «Мой GitLab»

[📊 Статистика сейчас]  [📜 Последние события]
[📅 Расписание]          [🔔 Типы событий]
[👥 Участники команды]   [🔗 Мой GitLab]
```

### 📅 Расписание

Позволяет изменить время ежедневного отчёта.

1. Нажмите **📅 Расписание**
2. Бот спросит новое время
3. Введите в формате `ЧЧ:ММ`, например `09:00` или `18:30`
4. Расписание обновится немедленно без перезапуска

> Настройка сохраняется в `bot.db` и восстанавливается при перезапуске бота.

### 🔗 Мой GitLab

Привязывает ваш GitLab username к Telegram-аккаунту.

1. Нажмите **🔗 Мой GitLab**
2. Введите ваш GitLab username (тот, что в URL профиля GitLab)
3. Готово — бот будет @упоминать вас при событиях от вашего аккаунта

**Пример:**
- GitLab профиль: `https://gitlab.com/ivan_petrov`
- Вводите: `ivan_petrov`

## Ежедневный отчёт

Бот автоматически отправляет ежедневный отчёт в настроенное время.

По умолчанию отчёт приходит в **09:00 (Московское время)** и содержит:
- Коммиты по каждому разработчику с @упоминанием
- Статистику CI/CD пайплайнов (успешные / упавшие)
- Активность по Merge Requests
- Активность по Issues
- Лидер дня по числу коммитов

Изменить время: `/tgbot` → **📅 Расписание**.
